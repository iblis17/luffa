=====
Luffa
=====

Project Luffa (絲瓜) aimed to provide integration between multiple Slack domains.

Installation
------------
::

 $ pip install git+https://github.com/rschiang/luffa.git

Run server
-----------

With bottle builtin server::

 $ luffa
